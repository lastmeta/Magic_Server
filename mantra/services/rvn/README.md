## Install Nest CLI
```bash
npm install -g @nestjs/cli
```

## Create a new Nest Project 
```bash
nest new my-project
```

## Initialize and Install Packages with BUN
```bash
bun init
bun install
```

## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Project setup

```bash
$ bun install
```

## Compile and run the project

```bash
# development
$ bun run start

# watch mode
$ bun run start:dev

# production mode
$ bun run start:prod
```

## Run tests

```bash
# unit tests
$ bun run test

# e2e tests
$ bun run test:e2e

# test coverage
$ bun run test:cov
```
